**Key Features**

Very simple application where users can order one or more pizzas from a menu

No user accounts or login required ‚Äî users just input their names before using the app

Pizza menu is dynamic and loaded from an API

Users can add multiple pizzas to a cart before ordering

Ordering requires only the user's name, phone number, and address

GPS location can be added to make delivery easier

Users can mark their order as priority, adding 20% to the total price

Orders are submitted via a POST request with user and pizza data

Payments are made on delivery ‚Äî no payment processing required

Each order receives a unique ID so users can check order status later

Users can still set an existing order as priority after placing it
